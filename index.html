_R

_R is a powerful JavaScript reflection library. _R supports inspecting functions, objects, lamba lifting, creating Proxies, subclassing arrays, securing objects, and partial support for ES6 Reflect object.

Citron

Citron is an alpha-stage matrix library for JavaScript. Citron utilises hand-written asm.js and extremely simple macro system.

Najtingalo

Najtingalo is an optimizing Brainfuck to JavaScript transpiler. It's goal is to be as effective as Esotope Brainfuck compiler.
demo

Dewormer

Dewormer is a simple AST-transforming tool designed to debug any environment compatible with ES3, especially these without debugger statement. It can be easily extended to allow remote debugging.
demo

Chat

This simple chat was written in 2012 in procedural style and was ported to Laravel in 2013. It supports multiple channels, emoticons, advanced front-end configuration.

extend-arguments

extend-arguments allow developer to use virtual prototype of arguments object.
